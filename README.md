🔍 AI Article Summarizer Chrome Extension : 
A professional Chrome Extension that instantly summarizes articles, blogs, and news content on any website using Google’s Gemini API. This project showcases your skills in Chrome Extension development, AI API integration, and modern UI design — making it a standout addition to your portfolio.

🚀 Features
✨ One-Click Summarization
Summarize articles, blogs, and news from any webpage instantly.

📝 Summary Modes
Choose between Brief, Detailed, or Bullet Point formats.

🤖 Gemini API Integration
High-quality, AI-generated summaries via Google’s Gemini API.

📋 Copy to Clipboard
Easily copy summaries for note-taking or sharing.

🔐 Secure API Key Management
API key is stored securely via the extension’s Options page.

🖌️ Modern & Responsive UI
Sleek, clean popup interface for a smooth user experience.

🛠️ Tech Stack
HTML5

CSS3

JavaScript (ES6+)

Chrome Extensions (Manifest V3)

Google Gemini API

⚡ Getting Started
1. Clone the Repository
<code> ```bash
git clone https://github.com/your-username/ai-article-summarizer.git
2. Get a Gemini API Key
Sign up at Google AI Studio

Generate a Gemini API key

3. Configure the Extension
Open options.html inside the extension

Enter your API key securely

4. Load Extension in Chrome
Open chrome://extensions/

Enable Developer mode (top right corner)

Click Load unpacked

Select the cloned project folder ```</code>
